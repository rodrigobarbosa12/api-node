## 🧩 Instalação

### Baixe o projeto
    $ git clone git@github.com:rodrigobarbosa12/api-node.git

### Instale todas as dependências
    $ yarn

### Execute localmente
    $ yarn dev

## TypeORM

### Migrações
    $ typeorm migration:create -n nome_da_migracao
    $ yarn migration:run
    $ yarn migration:revert
